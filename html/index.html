Espresso

Espresso - A web framework that compatible to C++14 and above. Espresso+ compiler which combines espresso code and C++ and converts to executable code, c++/stl headers can easily be included in espresso source files.

This web framework accept callback function as an argument. Distributed to apt repository from where can be installed on Linux system.
